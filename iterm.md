# Settings for iterm2

* Keybindings
- cmd -> move tabs
* Shortcuts 
- Cmd t -> new tab

* Font:
- Iosevka Term Slab
- For Mac, download from here: https://github.com/be5invis/Iosevka/releases
- Install and add in iterm2 settings -> profiles -> text.
- Use the extended version with fontsize `17` and v|i `98`.

* Rose pine moon:
- Download from here: https://rosepinetheme.com/themes/
- Add in iterm2 settings -> profiles -> theme
